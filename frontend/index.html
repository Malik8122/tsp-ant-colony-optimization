AS, MMAS, and Rank-Based AS — Comparative Study

Abstract. This interface implements and compares three ant-based metaheuristics for the 5-city TSP. Standard AS uses all ants for pheromone updates. MMAS restricts updates to the best ant with pheromone bounding. Rank-Based AS (AS-rank) uses a weighted update from the top-ranked ants and the best-so-far solution to guide discovery.
